# qyNews
Android平台新闻客户端

一、需求分析：大众对于获取新闻的需求一直存在：报纸 > 电视新闻联播 > 门户网站 > 手机新闻客户端


二、项目设计: 1.新闻：提供新闻 2.我的：展示个人主页 3.入口：登录注册
